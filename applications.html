Real-World Applications

Apply derivatives to solve practical problems in physics, economics, optimization, and related rates - bridging mathematics to the real world

1. Related Rates

Related rates problems involve finding the rate at which one quantity changes by relating it to other quantities whose rates of change are known.

Problem-Solving Strategy:

1. Identify all relevant quantities and assign variables

2. Find an equation relating the variables (often geometry)

3. Differentiate implicitly with respect to time (t)

4. Substitute known values and solve for the unknown rate

Example 1: Expanding Balloon

A spherical balloon is being inflated at a rate of 100 cm³/s. How fast is the radius increasing when the radius is 5 cm?

Given: dV/dt = 100 cm³/s, r = 5 cm

Find: dr/dt

Step 1: Relate V and r using volume formula:

V = (4/3)πr³

Step 2: Differentiate with respect to t:

dV/dt = 4πr² · dr/dt

Step 3: Solve for dr/dt:

100 = 4π(5)² · dr/dt

100 = 100π · dr/dt

dr/dt = 1/π cm/s

∴ The radius is increasing at 1/π cm/s ≈ 0.318 cm/s

Example 2: Ladder Sliding Down Wall

A 10-meter ladder rests against a wall. If the bottom slides away at 2 m/s, how fast is the top sliding down when the bottom is 6 m from the wall?

Given: dx/dt = 2 m/s (bottom sliding out), L = 10 m, x = 6 m

Find: dy/dt (top sliding down)

Step 1: Pythagorean theorem: x² + y² = L²

Step 2: Find y when x = 6:

y² = 100 - 36 = 64, so y = 8 m

Step 3: Differentiate: 2x · dx/dt + 2y · dy/dt = 0

2(6)(2) + 2(8) · dy/dt = 0

24 + 16 · dy/dt = 0

dy/dt = -24/16 = -3/2 m/s

∴ The top slides down at 1.5 m/s

2. Optimization Problems

Optimization involves finding the maximum or minimum value of a function under given constraints - a critical tool in economics, engineering, and business.

Problem-Solving Strategy:

1. Identify the quantity to maximize or minimize (objective function)

2. Express it as a function of a single variable

3. Find the domain of this function

4. Find critical points (where f'(x) = 0 or undefined)

5. Determine endpoints and critical points, evaluate to find max/min

6. Verify the result makes sense in context

Example 3: Maximizing Area

A farmer has 200 m of fencing to enclose a rectangular area. What dimensions maximize the area?

Step 1: Define variables

Let x = width, y = length

Constraint: 2x + 2y = 200 → x + y = 100

Area: A = xy

Step 2: Express A as function of x only

y = 100 - x

A(x) = x(100 - x) = 100x - x²

Step 3: Find critical points

A'(x) = 100 - 2x = 0

x = 50 m

Step 4: Verify maximum

A''(x) = -2 < 0 (concave down → maximum)

y = 100 - 50 = 50 m

∴ Maximum area is 50m × 50m = 2500 m²

Example 4: Minimizing Cost

A cylindrical can is to hold 1000 cm³. The material for the top and bottom costs twice as much per cm² as the side. Find the dimensions that minimize cost.

Step 1: Define variables

r = radius, h = height

Volume: πr²h = 1000 → h = 1000/(πr²)

Step 2: Cost function

Cost(top+bottom) = 2 · 2πr² = 4πr² (twice the material cost)

Cost(side) = 1 · 2πrh = 2πrh

Total: C = 4πr² + 2πrh

Step 3: Substitute h and find C(r)

C(r) = 4πr² + 2πr(1000/(πr²)) = 4πr² + 2000/r

Step 4: Minimize C(r)

C'(r) = 8πr - 2000/r² = 0

8πr³ = 2000

r³ = 250/π

r ≈ 4.3 cm, h ≈ 17.2 cm

∴ Optimal dimensions: r ≈ 4.3 cm, h ≈ 17.2 cm

3. Motion and Kinematics

Derivatives describe motion: position's first derivative is velocity, second derivative is acceleration. This is fundamental in physics and engineering.

Key Relationships:

• Position: s(t)

• Velocity: v(t) = s'(t) (rate of change of position)

• Acceleration: a(t) = v'(t) = s''(t) (rate of change of velocity)

• Speed = |v(t)| (always non-negative)

Example 5: Analyzing Motion

A particle moves along a line with position s(t) = t³ - 6t² + 9t + 1 (meters). Find:

(a) Velocity and acceleration functions

(b) When the particle is at rest

(c) Total distance traveled in first 5 seconds

(a) Velocity and Acceleration:

v(t) = s'(t) = 3t² - 12t + 9

a(t) = v'(t) = 6t - 12

(b) When at rest (v = 0):

3t² - 12t + 9 = 0

t² - 4t + 3 = 0

(t-1)(t-3) = 0

t = 1 s or t = 3 s

(c) Total distance:

Find position at key times: s(0) = 1, s(1) = 5, s(3) = 1, s(5) = 21

Movement: 0→1: +4m, 1→3: -4m, 3→5: +20m

Total distance = 4 + 4 + 20 = 28 m

Example 6: Economics Application - Profit Maximization

A company's profit function is P(x) = -2x² + 40x - 100 (in thousands of dollars), where x is units produced (in hundreds). Find the maximum profit and optimal production level.

Step 1: Find critical points

P'(x) = -4x + 40 = 0

x = 10 (hundreds of units = 1000 units)

Step 2: Verify maximum

P''(x) = -4 < 0 (concave down → maximum)

Step 3: Calculate maximum profit

P(10) = -2(100) + 40(10) - 100 = 100

∴ Maximum profit is $100,000 at 1000 units

4. Curve Sketching with Derivatives

Derivatives help us understand the shape and behavior of graphs, identifying extrema, inflection points, and monotonicity.

Key Information from Derivatives:

• f'(x) > 0: function is increasing

• f'(x) < 0: function is decreasing

• f'(x) = 0: potential local extrema

• f''(x) > 0: function is concave up

• f''(x) < 0: function is concave down

• f''(x) = 0: potential inflection point

Example 7: Complete Curve Analysis

Analyze and sketch f(x) = x³ - 3x² - 9x + 10

Step 1: First derivative (critical points)

f'(x) = 3x² - 6x - 9 = 3(x² - 2x - 3) = 3(x-3)(x+1)

Critical points: x = -1, x = 3

Step 2: Second derivative (inflection)

f''(x) = 6x - 6 = 0 → x = 1

Inflection point at x = 1

Step 3: Classification

x < -1: f'(x) > 0 (increasing)

-1 < x < 3: f'(x) < 0 (decreasing)

x > 3: f'(x) > 0 (increasing)

Local max at x = -1, local min at x = 3

Step 4: End behavior

As x → ∞, f(x) → ∞; as x → -∞, f(x) → -∞
