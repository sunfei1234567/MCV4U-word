Limits & Continuity

Foundation of Calculus: Understanding limits, limit existence theorem, and the relationship between continuity and differentiability

1. Understanding Limits

The concept of a limit is fundamental to calculus. It describes the behavior of a function as its input approaches a particular value.

lim (x→a) f(x) = L
"The limit of f(x) as x approaches a equals L"

Consider the function f(x) = (x² - 4)/(x - 2). Direct substitution at x = 2 gives 0/0, which is undefined. However, the limit exists because we can simplify and analyze the behavior as x approaches 2 from both sides.

Example 1: Finding Limits by Simplification

Find: lim (x→2) (x² - 4)/(x - 2)

Solution:

Step 1: Factor the numerator: x² - 4 = (x + 2)(x - 2)

Step 2: Simplify: (x + 2)(x - 2)/(x - 2) = x + 2 (for x ≠ 2)

Step 3: Evaluate the limit: lim (x→2) (x + 2) = 4

∴ The limit equals 4

2. Limit Existence Theorem

A limit exists if and only if the left-hand limit and right-hand limit are equal.

lim (x→a) f(x) = L ⟺ lim (x→a⁻) f(x) = lim (x→a⁺) f(x) = L
Left-hand limit must equal right-hand limit

Key Theorem: Limits and Function Values

The existence of a limit does NOT depend on whether f(a) is defined. The function can be defined or undefined at x = a, but if the limit exists, it is still valid.

Example 2: One-Sided Limits

Determine if the limit exists: f(x) = { x + 2 if x < 1; 3 - x if x ≥ 1 } at x = 1

Solution:

Left-hand limit: lim (x→1⁻) f(x) = lim (x→1⁻) (x + 2) = 1 + 2 = 3

Right-hand limit: lim (x→1⁺) f(x) = lim (x→1⁺) (3 - x) = 3 - 1 = 2

Since 3 ≠ 2, the limit does NOT exist

3. Continuity

A function is continuous at a point x = a if three conditions are met:

f(a) is defined ∧ lim (x→a) f(x) exists ∧ lim (x→a) f(x) = f(a)
All three conditions must be satisfied

Types of Discontinuities:

• Removable: Hole at the point (can be "filled")

• Jump: Sudden change in value (left ≠ right)

• Infinite: Vertical asymptote (function grows without bound)

Example 3: Testing Continuity

Is f(x) = (x² - 9)/(x - 3) continuous at x = 3?

Solution:

1. f(3) = (9 - 9)/0 = 0/0 = undefined ✗

2. lim (x→3) (x² - 9)/(x - 3) = lim (x→3) (x + 3) = 6 (limit exists)

Since f(3) is not defined, f is NOT continuous at x = 3

However, if we define g(x) = x + 3 (with g(3) = 6), then g IS continuous at x = 3

4. Relationship: Continuity & Differentiability

Critical Theorem

If a function is differentiable at a point, then it must be continuous at that point. However, continuity does NOT guarantee differentiability.

⚠️ Important: All differentiable functions are continuous, but not all continuous functions are differentiable. A classic example is f(x) = |x| at x = 0, which is continuous but not differentiable (the left and right derivatives don't match).

Example 4: Continuity ≠ Differentiability

Show that f(x) = |x| is continuous at x = 0 but not differentiable there.

Continuity:

• f(0) = |0| = 0 (defined)

• lim (x→0) |x| = 0 (exists)

• lim (x→0) |x| = f(0) = 0 ✓

∴ f is continuous at x = 0

Differentiability:

• Left derivative: lim (h→0⁻) (|0 + h| - |0|)/h = lim (h→0⁻) (-h)/h = -1

• Right derivative: lim (h→0⁺) (|0 + h| - |0|)/h = lim (h→0⁺) (h)/h = 1

Since -1 ≠ 1, f is NOT differentiable at x = 0

5. Special Limit Rules

lim (x→0) sin(x)/x = 1
Squeeze Theorem corollary
lim (x→0) (1 - cos(x))/x = 0
Using trig identity
lim (x→∞) (1 + 1/x)^x = e ≈ 2.71828
Definition of e

Example 5: Trigonometric Limit

Find: lim (x→0) sin(3x)/x

Solution:

Using substitution and the known limit:

lim (x→0) sin(3x)/x = lim (x→0) [sin(3x)/(3x)] × 3

= 1 × 3 = 3

∴ The limit equals 3
