Limits and Continuity

Evaluate limits using various techniques including direct substitution, factorization, rationalization, and the squeeze theorem.

Limits Continuity One-sided Limits Squeeze Theorem
Assignment 1 Solutions

Question 1: Evaluate lim (x→2) (x² - 4)/(x - 2)

Factor: (x² - 4)/(x - 2) = (x+2)(x-2)/(x-2) = x + 2 (for x ≠ 2)

lim (x→2) (x + 2) = 4

Answer: 4

Question 2: Evaluate lim (x→0) sin(3x)/x

Using the known limit lim (x→0) sin(x)/x = 1:

lim (x→0) sin(3x)/x = lim (x→0) [sin(3x)/(3x)] × 3 = 1 × 3 = 3

Answer: 3

Question 3: Determine if f(x) = {x² if x < 1; 2x - 1 if x ≥ 1} is continuous at x = 1

f(1) = 2(1) - 1 = 1

lim (x→1⁻) x² = 1, lim (x→1⁺) (2x - 1) = 1

Since all three conditions are met, f is continuous at x = 1

Answer: Continuous

Assignment 2 Solutions

Question 1: Find f'(x) if f(x) = 3x² + 2x - 5 using the definition

f'(x) = lim (h→0) [f(x+h) - f(x)]/h

= lim (h→0) [3(x+h)² + 2(x+h) - 5 - (3x² + 2x - 5)]/h

= lim (h→0) [3x² + 6xh + 3h² + 2x + 2h - 5 - 3x² - 2x + 5]/h

= lim (h→0) [6xh + 3h² + 2h]/h = lim (h→0) (6x + 3h + 2) = 6x + 2

Answer: f'(x) = 6x + 2

Question 2: Find the second derivative of f(x) = x⁴ - 3x²

f'(x) = 4x³ - 6x

f''(x) = 12x² - 6

Answer: f''(x) = 12x² - 6

Derivative Rules - Power, Product, Quotient

Master the three essential rules for differentiation: power rule, product rule, and quotient rule through comprehensive practice.

Power Rule Product Rule Quotient Rule Combined Rules
Assignment 3 Solutions

Question 1: Differentiate f(x) = (2x³ + 1)(x² - 3x)

Using product rule: f'(x) = (2x³ + 1)'(x² - 3x) + (2x³ + 1)(x² - 3x)'

= (6x²)(x² - 3x) + (2x³ + 1)(2x - 3)

= 6x⁴ - 18x³ + 4x⁴ - 6x³ + 2x - 3

= 10x⁴ - 24x³ + 2x - 3

Answer: 10x⁴ - 24x³ + 2x - 3

Question 2: Differentiate f(x) = (x² + 4)/(3x - 1)

Using quotient rule: f'(x) = [(2x)(3x-1) - (x²+4)(3)]/(3x-1)²

= [6x² - 2x - 3x² - 12]/(3x-1)²

= [3x² - 2x - 12]/(3x-1)²

Answer: (3x² - 2x - 12)/(3x-1)²

Question 1: Find the derivative of f(x) = sin(5x² + 2x)

Using chain rule: f'(x) = cos(5x² + 2x) × d/dx(5x² + 2x)

= cos(5x² + 2x) × (10x + 2)

Answer: (10x + 2)cos(5x² + 2x)

Question 2: Differentiate f(x) = (tan(x))⁴

Using chain rule: f'(x) = 4(tan(x))³ × sec²(x)

Answer: 4tan³(x)sec²(x)

Assignment 5 Solutions

Question 1 (Related Rates): A ladder 13m long leans against a wall. If the bottom slides away at 5 m/s, how fast is the top sliding down when the bottom is 5m from the wall?

x² + y² = 169 (Pythagorean theorem)

When x = 5: y = √(169-25) = √144 = 12m

Differentiate: 2x(dx/dt) + 2y(dy/dt) = 0

2(5)(5) + 2(12)(dy/dt) = 0

50 + 24(dy/dt) = 0

dy/dt = -50/24 = -25/12 m/s

Answer: -25/12 m/s (≈ -2.08 m/s)

Question 2 (Optimization): Find the maximum area of a rectangle with perimeter 40cm.

2x + 2y = 40 → x + y = 20 → y = 20 - x

A = xy = x(20-x) = 20x - x²

A'(x) = 20 - 2x = 0 → x = 10cm

y = 20 - 10 = 10cm

Maximum area = 10 × 10 = 100 cm²

Answer: 100 cm² (when rectangle is a square)
