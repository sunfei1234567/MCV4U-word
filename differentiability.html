Differentiability

Understanding Derivatives: The instantaneous rate of change, derivative definitions, and the relationship between differentiability and continuity

1. The Derivative Definition

The derivative of a function represents the instantaneous rate of change at a specific point. It can be defined in multiple equivalent ways.

f'(x) = lim (h→0) [f(x + h) - f(x)] / h
Two-point difference quotient (limit form)
f'(a) = lim (x→a) [f(x) - f(a)] / (x - a)
Alternative form using x → a

Example 1: Finding Derivative from Definition

Find the derivative of f(x) = x² using the definition.

Solution:

f'(x) = lim (h→0) [(x + h)² - x²] / h

= lim (h→0) [x² + 2xh + h² - x²] / h

= lim (h→0) [2xh + h²] / h

= lim (h→0) [2x + h]

∴ f'(x) = 2x

2. Differentiability and Continuity

Fundamental Relationship

If f is differentiable at x = a, then f is continuous at x = a.

Proof: If f'(a) exists, then lim (x→a) f(x) = f(a), which is the definition of continuity.

⚠️ Key Insight: Differentiability ⟹ Continuity (but NOT the reverse!)

When is a function NOT differentiable at a point?

• Sharp corners (like |x| at x = 0)

• Cusps (pointed vertices)

• Vertical tangents (like x^(1/3) at x = 0)

• discontinuities

Example 2: Testing Differentiability

Is f(x) = { x² if x < 2; 4x - 4 if x ≥ 2 } differentiable at x = 2?

Step 1: Check continuity

• f(2) = 4(2) - 4 = 4

• lim (x→2⁻) x² = 4

• lim (x→2⁺) (4x - 4) = 4

∴ f is continuous at x = 2 ✓

Step 2: Check differentiability (left and right derivatives)

• f'_+(2) = lim (h→0⁺) [(4(2+h) - 4) - 4] / h = 4

• f'_-(2) = lim (h→0⁻) [(2+h)² - 4] / h = 4

Since both derivatives equal 4, f is differentiable at x = 2

3. The Leibniz Notation

Derivatives can be expressed in multiple notations:

dy/dx = f'(x) = D[f(x)]
Leibniz, Prime, and Differential operator notation
d/dx [f(x)] = lim (Δx→0) Δy/Δx
Meaning: "ratio of infinitesimal change in y to change in x"
⚠️ Important: dy/dx is NOT a fraction! It cannot be split apart. However, in certain contexts (related rates, separation of variables), we treat it as if it could be manipulated algebraically.

4. Higher Order Derivatives

The derivative of a derivative is called a higher order derivative.

f''(x) = d/dx [f'(x)]
Second derivative (rate of change of rate of change)
f'''(x), f⁽⁴⁾(x), ... , f⁽ⁿ⁾(x)
Third, fourth, and nth derivatives

Example 3: Finding Higher Order Derivatives

Find f''(x) if f(x) = 4x³ + 2x² - 3x + 7

Solution:

f'(x) = 12x² + 4x - 3

f''(x) = d/dx [12x² + 4x - 3] = 24x + 4

f'''(x) = d/dx [24x + 4] = 24

f⁽⁴⁾(x) = d/dx [24] = 0

∴ f''(x) = 24x + 4

5. Physical Interpretation

If s(t) represents position at time t:

• v(t) = s'(t) = ds/dt (velocity - rate of change of position)

• a(t) = v'(t) = s''(t) = d²s/dt² (acceleration - rate of change of velocity)

• j(t) = a'(t) = s'''(t) (jerk - rate of change of acceleration)

Example 4: Physics Application

A ball is thrown upward with position s(t) = 50t - 5t² meters. Find velocity and acceleration at t = 3 seconds.

Solution:

v(t) = s'(t) = 50 - 10t

a(t) = s''(t) = -10 m/s²

v(3) = 50 - 10(3) = 20 m/s (still moving upward)

a(3) = -10 m/s² (gravity pulling down)
