Derivative Rules

Essential differentiation techniques: Power Rule, Product Rule, Quotient Rule, and Chain Rule - the essential toolkit for derivatives

1. Power Rule

The most fundamental rule for differentiating polynomial and power functions.

d/dx [xⁿ] = nxⁿ⁻¹
For any real number n

Special Cases:

• d/dx [x] = 1 (when n = 1)

• d/dx [constant] = 0

• d/dx [√x] = d/dx [x^(1/2)] = (1/2)x^(-1/2) = 1/(2√x)

Example 1: Power Rule Applications

Find the derivative of f(x) = 5x⁴ + 3x² - 2x + 7

Solution:

f'(x) = 5 · 4x³ + 3 · 2x - 2 + 0

= 20x³ + 6x - 2

∴ f'(x) = 20x³ + 6x - 2

2. Product Rule

When differentiating the product of two functions, use this rule.

d/dx [f(x) · g(x)] = f'(x) · g(x) + f(x) · g'(x)
"First times derivative of second, plus second times derivative of first"

Memory Aid: (f · g)' = f' · g + f · g'

Alternative notation: (uv)' = u'v + uv'

Example 2: Product Rule

Find the derivative of f(x) = (3x² + 1)(x³ - 2x)

Solution:

Let u = (3x² + 1) and v = (x³ - 2x)

u' = 6x and v' = 3x² - 2

f'(x) = (6x)(x³ - 2x) + (3x² + 1)(3x² - 2)

= 6x⁴ - 12x² + 9x⁴ - 6x² + 3x² - 2

= 15x⁴ - 15x² - 2

∴ f'(x) = 15x⁴ - 15x² - 2

3. Quotient Rule

When differentiating a quotient of two functions, use this rule.

d/dx [f(x)/g(x)] = [f'(x)·g(x) - f(x)·g'(x)] / [g(x)]²
"Low d-high minus high d-low, over low squared"

Memory Aid: (f/g)' = (f'g - fg')/g²

Alternative notation: (u/v)' = (u'v - uv')/v²

⚠️ Common Mistake: Do NOT simply differentiate numerator and denominator separately. The quotient rule is required.

Example 3: Quotient Rule

Find the derivative of f(x) = (2x² + 3)/(x - 1)

Solution:

Let u = 2x² + 3, v = x - 1

u' = 4x, v' = 1

f'(x) = [(4x)(x - 1) - (2x² + 3)(1)] / (x - 1)²

= [4x² - 4x - 2x² - 3] / (x - 1)²

= [2x² - 4x - 3] / (x - 1)²

∴ f'(x) = (2x² - 4x - 3)/(x - 1)²

4. Chain Rule

When differentiating composite functions (functions of functions), use the chain rule.

d/dx [f(g(x))] = f'(g(x)) · g'(x)
"Outer derivative times inner derivative"

Step-by-step process:

1. Identify the outer function f and inner function g

2. Find f'(u) and g'(x)

3. Multiply: f'(g(x)) · g'(x)

4. Substitute back to original variable

Example 4: Chain Rule

Find the derivative of f(x) = (3x² + 2)⁵

Solution:

Outer function: f(u) = u⁵

Inner function: u = g(x) = 3x² + 2

f'(u) = 5u⁴, g'(x) = 6x

f'(x) = 5(3x² + 2)⁴ · 6x

= 30x(3x² + 2)⁴

∴ f'(x) = 30x(3x² + 2)⁴

Example 5: Chain Rule with Multiple Applications

Find the derivative of f(x) = sin(4x³)

Solution:

Outer: sin(u), Inner: u = 4x³

d/dx[sin(u)] = cos(u) · u'

f'(x) = cos(4x³) · 12x²

∴ f'(x) = 12x²cos(4x³)

5. Trigonometric Derivatives

Essential derivatives for trigonometric functions.

d/dx [sin(x)] = cos(x)
Sine derivative
d/dx [cos(x)] = -sin(x)
Cosine derivative
d/dx [tan(x)] = sec²(x)
Tangent derivative

Example 6: Trigonometric Derivatives

Find the derivative of f(x) = 3sin(x) - 2cos(x) + tan(x)

Solution:

f'(x) = 3cos(x) - 2(-sin(x)) + sec²(x)

= 3cos(x) + 2sin(x) + sec²(x)

∴ f'(x) = 3cos(x) + 2sin(x) + sec²(x)

6. Exponential and Logarithmic Derivatives

d/dx [eˣ] = eˣ
Natural exponential (e ≈ 2.71828)
d/dx [aˣ] = aˣ · ln(a)
General exponential
d/dx [ln(x)] = 1/x
Natural logarithm
d/dx [logₐ(x)] = 1/(x · ln(a))
General logarithm

Example 7: Exponential and Logarithmic Derivatives

Find the derivative of f(x) = x² · eˣ · ln(x)

Solution: This requires multiple rules (product rule + chain rule)

This is a product of three functions: x², eˣ, and ln(x)

Using extended product rule:

f'(x) = (2x · eˣ · ln(x)) + (x² · eˣ · ln(x)) + (x² · eˣ · 1/x)

= 2xeˣln(x) + x²eˣln(x) + xeˣ

= eˣ[2xln(x) + x²ln(x) + x]
