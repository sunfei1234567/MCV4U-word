Question 1

State the formal definition of a derivative. Explain the difference between the derivative and the slope of a secant line.

Show Solution

Formal Definition of Derivative:

The derivative of f(x) is defined as:

f'(x) = lim (h→0) [f(x + h) - f(x)] / h

Difference between Derivative and Secant Slope:

• Secant line slope: m = [f(x₂) - f(x₁)] / (x₂ - x₁) - uses two distinct points

• Derivative: f'(x) = lim (h→0) [f(x + h) - f(x)] / h - uses the limit as h approaches 0

• The derivative is the instantaneous rate of change at a specific point

• The secant slope is the average rate of change over an interval

• When x₂ → x₁, the secant slope approaches the derivative

Question 2

List and explain the three conditions required for a function to be continuous at a point x = a.

Show Solution

Three Conditions for Continuity at x = a:

For f(x) to be continuous at x = a, ALL three conditions must be satisfied:

1. f(a) is defined - The function must have a value at x = a (no hole or gap)

2. lim (x→a) f(x) exists - The limit as x approaches a must exist (left limit = right limit)

3. lim (x→a) f(x) = f(a) - The limit value must equal the function value at that point

Example: f(x) = x² at x = 2

• f(2) = 4 (defined) ✓

• lim (x→2) x² = 4 (exists) ✓

• 4 = 4 (equal) ✓

∴ f(x) = x² is continuous at x = 2

Question 3 - Related Rates

A conical tank with height 10m and radius 3m is being filled with water at a rate of 2 m³/min. How fast is the water level rising when the water is 6m deep?

Show Solution

Given:

• dV/dt = 2 m³/min (filling rate)

• H = 10m, R = 3m (tank dimensions)

• h = 6m (water depth at instant)

Find: dh/dt when h = 6m

Step 1: Relate V and h using similar triangles

R/H = r/h where r is water radius at height h

3/10 = r/h → r = (3/10)h

Step 2: Volume of cone

V = (1/3)πr²h = (1/3)π[(3/10)h]²h = (1/3)π(9/100)h³ = (3/100)πh³

Step 3: Differentiate with respect to t

dV/dt = (9/100)πh² · dh/dt

Step 4: Solve for dh/dt

2 = (9/100)π(6)² · dh/dt

2 = (9/100)π(36) · dh/dt

2 = (324/100)π · dh/dt

dh/dt = 200/(324π) = 50/(81π) m/min

Answer: dh/dt = 50/(81π) ≈ 0.196 m/min

Question 4 - Optimization

A rectangle field is to be enclosed on three sides by fencing, with one side along a river (no fencing needed). If 200m of fencing is available, find the dimensions that maximize the enclosed area.

Show Solution

Step 1: Define variables

• Let x = width (two sides perpendicular to river)

• Let y = length (side parallel to river)

• Constraint: 2x + y = 200 → y = 200 - 2x

Step 2: Write objective function

A = xy = x(200 - 2x) = 200x - 2x²

Step 3: Find critical points

A'(x) = 200 - 4x = 0

x = 50m

Step 4: Verify maximum

A''(x) = -4 < 0 (concave down → maximum)

y = 200 - 2(50) = 100m

Step 5: Calculate maximum area

A = 50 × 100 = 5000 m²

Answer: Width = 50m (×2), Length = 100m, Maximum area = 5000 m²

Question 5 - Proof

Prove that if a function f is differentiable at x = a, then f is continuous at x = a.

Show Solution

Proof:

Given: f is differentiable at x = a (i.e., f'(a) exists)

To Prove: f is continuous at x = a (i.e., lim (x→a) f(x) = f(a))

Proof:

Since f is differentiable at x = a:

f'(a) = lim (h→0) [f(a + h) - f(a)] / h exists

Consider the identity:

f(a + h) - f(a) = [f(a + h) - f(a)] / h × h

Taking the limit as h → 0:

lim (h→0) [f(a + h) - f(a)] = lim (h→0) [f(a + h) - f(a)] / h × lim (h→0) h

= f'(a) × 0 = 0

Therefore:

lim (h→0) f(a + h) - f(a) = 0

lim (h→0) f(a + h) = f(a)

Which is equivalent to:

lim (x→a) f(x) = f(a)

∎ Q.E.D.

Question 6 - Strategy Analysis

For f(x) = x³ - 3x + 1, analyze the behavior completely including intervals of increase/decrease, local extrema, and inflection points. Explain your strategy for this analysis.

Show Solution

Strategy:

1. Find f'(x) to determine increasing/decreasing intervals

2. Find critical points where f'(x) = 0 or undefined

3. Use first derivative test to classify critical points

4. Find f''(x) to determine concavity and inflection points

5. Set f''(x) = 0 to find potential inflection points

Complete Analysis:

Step 1: f'(x) = 3x² - 3 = 3(x² - 1) = 3(x-1)(x+1)

Critical points: x = -1, x = 1

Step 2: Test intervals

• x < -1: f'(x) > 0 (increasing)

• -1 < x < 1: f'(x) < 0 (decreasing)

• x > 1: f'(x) > 0 (increasing)

Step 3: Classify extrema

• x = -1: local maximum (changes from ↑ to ↓)

• x = 1: local minimum (changes from ↓ to ↑)

Step 4: f''(x) = 6x

Set f''(x) = 0: 6x = 0 → x = 0

Step 5: Test concavity

• x < 0: f''(x) < 0 (concave down)

• x > 0: f''(x) > 0 (concave up)

Inflection point at x = 0

Question 7 - Written Explanation

Explain the relationship between velocity and acceleration in calculus terms. Use a real-world example (such as a car braking) to illustrate how these concepts work together. Your explanation should demonstrate proper mathematical notation and clear reasoning.

Show Solution

Mathematical Relationship:

In calculus, motion along a line is described by position s(t). The instantaneous velocity v(t) is the first derivative of position:

v(t) = ds/dt

Acceleration a(t) is the first derivative of velocity (and second derivative of position):

a(t) = dv/dt = d²s/dt²

Real-World Example: Car Braking

Consider a car traveling at 30 m/s (≈108 km/h) that begins braking with constant deceleration.

• At t = 0: v(0) = 30 m/s, a(0) = -5 m/s² (negative indicates slowing down)

• At t = 2s: v(2) = 30 + (-5)(2) = 20 m/s

• At t = 6s: v(6) = 30 + (-5)(6) = 0 m/s (car stops)

Key Observations:

• Velocity decreases linearly while acceleration remains constant

• When v(t) = 0 and a(t) < 0, the car has stopped but would continue accelerating negatively if the brake remained applied

• The negative acceleration (deceleration) causes velocity to decrease toward zero

Connection to Derivatives:

The slope of the velocity-time graph equals acceleration. The area under the acceleration-time graph equals the change in velocity.
